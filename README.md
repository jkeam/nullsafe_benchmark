* Lodash v4.17.11
* Ramda v0.26.1

* MacBook Pro (15-inch, 2016)
* macOS High Sierra 10.13.4 (17E202)
* Processor 2.6 GHz Intel Core i7
* Memory 16 GB 2133 MHz LPDDR3
* Node v11.11.0
